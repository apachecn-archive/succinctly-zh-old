# 第 4 章箭头功能

`Arrows`是使用`=>`语法的功能简写。它们在语法上类似于 C# ，Java 和 CoffeeScript 中的胖箭头语法。它们支持表达式主体和返回表达式值的语句块主体。与函数不同，箭头与其周围代码共享相同的词法`this`。

以下是经典函数看起来像保持相同的词汇`this`：

代码 70

```
var square = (function(num)
  {
  return num * num;
}).bind(this);

```

现在，我们可以简单地写如下：

代码 71

```
let square = (num) =>
  {
  return num * num;
};

```
