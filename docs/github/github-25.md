## 摘要

GitHub 通过在搜索文本中找到的关键字来确定您正在使用的搜索类型。它会将关键字应用于所有适用的组（存储库，问题，代码，用户）。例如，创建的：关键字适用于存储库和用户，因此 GitHub 将返回在日期创建的存储库和在该日期加入的用户。
