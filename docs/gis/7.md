# 缩略语和缩略语

我认为留下你在整本书中看到的一些首字母缩略词列表是恰当的。像任何行业一样，有许多首字母缩略词需要熟悉，尽管我试图在可能的情况下解释它们，但摘要清单永远不会受到伤害。

EPSG

欧洲石油调查组

ESRI

环境系统研究所

EWKT

扩展着名的文字

GIS

地理信息系统

GML

地理标记语言

GPS

地理定位系统;地理定位卫星;全球定位系统

KML

锁孔标记语言

NMEA

国家海洋电子协会

OGC

开放地理空间联盟

OSGeo

开源地理空间基金会

SRID

空间参考标识符

WKB

众所周知的二进制文件，GIS 数据库中数据的二进制格式

WKT

众所周知的文本，GIS 数据库中数据的文本格式
